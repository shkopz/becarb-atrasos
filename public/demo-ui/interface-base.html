Resumen operativo

Lógica del beta
  • La hora se obtiene automáticamente del sistema.
  • El cruce se hace con la base demo de estudiantes.
  • Si el alumno ya fue registrado hoy, se bloquea el duplicado.
  • La alerta aparece cuando supera 3 atrasos en el mes.
  • El envío de correo está simulado en esta etapa.
  • Los reseteos individuales quedan registrados con fecha, hora y usuario.
  • El cierre mensual automático a inspectoría se simula en esta demo.
Los movimientos del mes siguen guardándose localmente en esta etapa, pero la nómina de alumnos ya se carga desde Google Sheets.
